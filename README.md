# 🗡️ Tilevania: 2D Cross-Platform Platformer

A full-loop, cross-platform 2D platformer developed for both PC and Android. Featuring intricate level design, dynamic touch-based mobile controls, and state-driven character animations.

> **🎬 Gameplay preview coming soon!** 🚧

## ✨ Core Mechanics & Features
* **📱 Cross-Platform Controls:** Engineered dynamic touch-based mobile joystick controls alongside standard PC keyboard inputs.
* **🧠 Advanced AI & Physics:** Programmed custom enemy AI behavior, projectile shooting mechanics, and environmental interactions using Unity Physics.
* **🎥 Dynamic Camera:** Integrated Unity's Cinemachine for smooth, responsive camera tracking and parallax background management.
* **🏃 State-Driven Animation:** Configured advanced Animator Controllers to handle complex sprite animations including running, climbing, jumping, and death states.
* **🖥️ Interactive UI:** Built interactive start screens and end-level menus utilizing TextMeshPro.
* **🗺️ Level Design:** Designed intricate, multi-layered levels utilizing Unity's TileMap and Rule Tiles system.

## 🛠️ Tech Stack
* **Game Engine:** Unity 2D
* **Language:** C#
* **Tools/Packages:** Unity Physics, Cinemachine, TextMeshPro, TileMap

## ⚙️ Local Setup Instructions
*(Note: Requires Unity Hub and Unity Editor to run locally)*

### 1. Clone the Repository
Open your terminal and run:
`git clone https://github.com/swapnilsensarma/Tilevania.git`

### 2. Open in Unity
* Open **Unity Hub**.
* Click **Add** or **Open** and select the cloned `Tilevania` folder.
* Allow Unity to resolve dependencies and rebuild the Library folder (this may take a few minutes on the first load).

### 3. Play the Game
* Navigate to the `Assets/Scenes` folder in the Project window.
* Open the Main Menu or Level 1 scene.
* Press the **Play** ▶️ button at the top of the editor to launch the game!
